I created Cable Planner because my brother-in-law explained what a difficult task it is for electricians to comply with Australian Standard wiring rules.

From there I got the idea to create Standards Updates so people can subscribe to get notifications when new Australian Standards are published. This gave me an opportunity to experiment with Google App Engine, which I really like.

I would like to spend more time on Cable Planner, to add features to it, and maybe rewrite what I’ve done so far to move it off WordPress, but that is a job that I don’t see myself having time for any time soon.
